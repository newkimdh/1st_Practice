 # My Toy Project

## 프로젝트 개요
이 프로젝트는 Git과 GitHub를 이용한 버전 관리 학습을 위한 토이 프로젝트입니다.

## 1주차 목표
- Git 및 GitHub의 기본적인 사용법 익히기
- 효율적인 폴더 구조 설계 및 관리

## 폴더 구조
my-toy-project/
├── public/      # 클라이언트 측(프론트엔드) 코드
└── server/      # 서버 측(백엔드) 코드
├── .gitignore   # Git 버전 관리에서 제외할 파일 및 폴더 지정
└── README.md    # 프로젝트 설명 문서
